# Machine Learning Study Guide

An interactive, high-fidelity web-based presentation and study tool covering **Chapter 10: Learning** (Sections 10.1 to 10.3.8) from the textbook *'Artificial Intelligence: Building Intelligent Systems'* by Parag Kulkarni and Prachi Joshi.

## 🌐 Live Link
**View the Site:** [https://yatharthg12.github.io/Machine-Learning-Basics/](https://yatharthg12.github.io/Machine-Learning-Basics/)

---

## 🎯 Project Purpose
This project was developed as a specialized seminar presentation tool. It transforms dense textbook theory into an engaging digital experience, making complex AI concepts accessible through interactive UI elements and visual storytelling.

## 📖 Key Topics Covered
The content is strictly aligned with the textbook curriculum:
* **Introduction to ML:** Goals, scope, and the challenge of data relevance.
* **Learning Paradigms:** Exploring search-based, observation-based, and problem-solving learning.
* **Learning Methods (The Core):**
    * **Rote Learning:** Direct knowledge acquisition.
    * **Supervised vs. Unsupervised:** Teacher-led learning vs. pattern discovery.
    * **Ensemble Learning:** The power of model combinations.
    * **Discovery-based Learning:** Data mining and KDD processes.
    * **Problem Solving:** Performance improvement through iterative task execution.

## ✨ Technical Features
* **Glassmorphism UI:** A modern, dark-themed aesthetic using CSS Backdrop-filters.
* **Interactive Quiz:** A built-in JavaScript assessment to test understanding post-presentation.
* **Dynamic Background:** Custom HTML5 Canvas particle system for a "neural network" visual effect.
* **Fully Responsive:** Optimized for projector screens, laptops, and mobile devices.
* **Code-Free Content:** Purely educational documentation designed for seminar delivery.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Custom Variables & Grid)
* **Logic:** Vanilla JavaScript (ES6+)
* **Design Framework:** Custom-built "Glass-AI" theme

---

## 📂 File Structure
- `index.html` — Contains the core educational content and semantic structure.
- `style.css` — Modern styling, animations, and responsive layout logic.
- `script.js` — Handles the particle engine, navigation, and quiz functionality.
